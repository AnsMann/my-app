# Status icons for diary entries

<Props of={EntryStatusIcons} />

**Status icons for every diary entry. Database dark means entry is in database. Database light means entry is only local. Trash icon means, entry will be deletetd next time user syncs with database**

  <br />
  <h3>Entry not databse</h3>
